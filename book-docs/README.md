# Physical AI and Humanoid Robotics: A Practical Guide

This directory contains the Docusaurus project for the "Physical AI and Humanoid Robotics: A Practical Guide" book.

## Development

To start the development server:

```bash
npm install
npm start
```

## Build

To build the static site:

```bash
npm install
npm run build
```

The built site will be located in the `build/` directory.
